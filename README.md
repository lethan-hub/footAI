# footAI

Motivation : 
- As an athlete and active individual, I have tried many different shoes and had to decide which are good and which are not. So with this app, individuals can easily access an application that measures their foot size and determines arch width, etc.

  
Target : 
- The primary target audience is people who shop for shoes online and want personalized, accurate recommendations without going in-store. This includes individuals across all activity levels — from runners and athletes to people who stand all day at work — who need footwear tailored to their specific use case (running, football, standing, etc.).
